# ADR-001: Локальная песочница PoC

## Статус

Принято

## Контекст

Нужно разработать демонстрационный PoC справочника без доступа к внутренней инфраструктуре, реальной Pulse UI и backend.

## Решение

- Использовать React 18 + TypeScript + Vite.
- Соблюдать frontend-only подход без backend и реальных интеграций.
- Подменить `@pulse/*` локальными stub-компонентами через alias.
- Использовать styled-components как основной способ описания UI.
- Сетевые сценарии моделировать локальным `fetch`-mock с переключением состояний через служебную панель.

## Последствия

- Можно разрабатывать интерфейс изолированно и воспроизводимо.
- В следующих итерациях потребуется постепенно расширять stubs и mock API, не ломая маршруты и базовую структуру проекта.
